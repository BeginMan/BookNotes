## 《Linux高性能服务器编程》读书笔记


![](http://img3.doubanio.com/lpic/s27327287.jpg)

[《Linux高性能服务器编程》](http://book.douban.com/subject/24722611/)

重点在第二篇，连带书籍：

- 《深入理解计算机系统》第十一章：网络编程和第十二章并发编程
- 《图解TCP/IP》
- 《计算机科学概论 第11版》第四章：组网及因特网
- 《Python网络编程攻略》
- 《Unix网络编程卷1.套接字联网API》
- 《TCP/IP详解 卷1：协议》

这些书籍看的断断续续的，有的看了一半，有的查阅了几下，如果2016年能把这些书籍看完也就很不错了。
